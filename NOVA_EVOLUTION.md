# Nova Evolution - v0.3.0 功能演进

这是 Nova 对 Vibe Todo 项目的天马行空的演进！✨

## 🚀 新功能概览

### 1. 🤖 AI 智能助手 (AI Helper)

新增完整的 AI 辅助模块，提供智能化的任务管理体验！

#### 核心功能

**📊 任务智能分析** (`ai analyze <task_id>`)
- 自动分析任务内容，推荐优先级
- 智能标签建议
- 预估工时
- 计算紧急程度分数
- 生成改进建议

**🎯 下一个任务推荐** (`ai next`)
- 综合考虑优先级、截止日期、依赖关系
- 短任务优先（快速成就感）
- 进行中任务优先（保持 momentum）
- 智能排序算法

**💡 上下文任务建议** (`ai suggest`)
- 基于时间段的智能建议（早晨/下午/晚上）
- 基于最近任务的模式匹配
- 保持项目 momentum 的建议

**📈 生产力评分** (`ai score [--days N]`)
- 综合生产力分数 (0-100)
- 完成率、工时、逾期惩罚
- 趋势分析和洞察

### 2. 🔗 任务依赖关系

任务模型新增 `depends_on` 字段，支持任务间的依赖关系！

```python
task = Task(
    title="完成项目",
    depends_on=["1", "2"]  # 依赖 ID 为 1 和 2 的任务
)
```

AI 推荐时会自动跳过依赖未完成的任务。

### 3. 🧠 AI 相关任务字段

任务模型新增 AI 相关字段：
- `ai_suggestions`: AI 生成的建议文本
- `ai_priority_score`: AI 计算的优先级分数 (0-1)

## 💻 CLI 新命令

### `ai` 命令组

所有 AI 功能都在 `ai` 命令组下：

```bash
# 分析任务
vibe ai analyze 1

# 推荐下一个任务
vibe ai next

# 获取任务建议
vibe ai suggest

# 查看生产力报告
vibe ai score
vibe ai score --days 14  # 查看 14 天的数据
```

## 🎨 核心算法

### 任务评分系统

AI 推荐时使用综合评分算法：

```
总分 = 优先级权重 + 紧急程度*50 + 短任务加分 + 进行中加分
```

- **优先级权重**: 紧急(100) > 高(75) > 中(50) > 低(25)
- **紧急程度**: 基于截止日期计算 (0-1)
- **短任务加分**: 新任务+10，<30分钟+5
- **进行中加分**: +30

### 模式匹配

内置多种任务模式识别：
- 会议模式 (meeting)
- 报告模式 (report)
- 邮件模式 (email)
- 代码模式 (code)
- 设计模式 (design)
- 个人模式 (personal)

根据关键词自动匹配，推荐对应的优先级、标签和预估工时。

### 时间段感知

根据一天中的不同时间给出不同建议：

**早晨 (5:00-12:00)**
- 查看今日待办
- 处理邮件

**下午 (12:00-18:00)**
- 专注开发（专注力最佳）

**晚上 (18:00-5:00)**
- 今日总结
- 反思回顾

## 📐 架构设计

### 文件结构

```
src/vibe_todo/
├── core/
│   ├── models.py          # 新增依赖和 AI 字段
│   ├── ai_helper.py       # ✨ 新增 AI 辅助模块
│   └── __init__.py        # 导出 AI 相关类
└── cli/
    └── main.py            # 新增 ai 命令组
```

### 核心类

**`AIHelper`** - 主 AI 助手类
- `analyze_task()`: 分析单个任务
- `suggest_next_task()`: 推荐下一个任务
- `generate_task_suggestions()`: 生成建议列表
- `calculate_productivity_score()`: 计算生产力分数

**`TaskSuggestion`** - 任务建议数据类
- title, description, priority, tags
- confidence (置信度)
- reason (推荐理由)

## 🎯 使用示例

### 1. 智能分析任务

```bash
# 添加一个任务
vibe add "完成项目文档" -p medium --due 2026-03-05

# AI 分析
vibe ai analyze 1
```

输出示例：
```

当前优先级: medium
截止日期: 2026-03-05

预估工时: 90 分钟
紧急程度: 50%

💡 建议:
  • 建议添加标签: 文档, 工作
```
